================================================================================
                        个人云盘系统 - 使用说明
                        （纯 PHP + JSON 存储，无需数据库）
================================================================================

一、系统简介
------------
这是一款基于纯 PHP 和 JSON 文件存储的轻量级个人云盘系统，无需 MySQL。
支持多用户注册/登录、文件上传下载、文件夹管理、分享链接（自定义有效期）、
图床功能（图片自动生成 Markdown 链接）、批量操作、在线预览等。
PC 和手机自适应，适合个人或小团队搭建私有云盘。


二、主要功能列表
----------------
【用户系统】
  - 用户注册（密码加密存储）
  - 用户登录（Session 保持，30 天自动登录）
  - 用户数据存于 users.json，文件隔离在 uploads/用户名/ 下

【文件与文件夹管理】
  - 文件上传：批量上传，自动重名（加 _1、_2 等后缀）
  - 文件下载：单个下载，批量下载为 ZIP 压缩包
  - 文件重命名、删除（单个或批量）
  - 文件移动（单个或批量，可跨目录）
  - 文件夹创建、重命名、删除（递归删除）

【分享与链接】
  - 生成分享链接：可自定义有效期（1天/3天/7天/30天/永久）
  - 撤销分享：已分享的文件可随时撤销，链接立即失效
  - 复制文件直接链接（可用于外链、图床）
  - 图片复制链接时自动提供 Markdown 格式选项

【图床功能】
  - 上传图片后复制链接即可用于博客、论坛
  - 支持 JPG / PNG / GIF / WebP / BMP 等格式
  - 链接格式：https://你的域名/uploads/用户名/图片路径

【批量操作】
  - 批量删除、批量移动、批量下载（ZIP）、批量复制链接
  - 支持全选/取消全选

【存储空间统计】
  - 实时显示当前用户已用空间（自动换算单位）
  - 仅统计该用户自己的文件

【转存功能】
  - 访问他人分享链接时，可一键转存到自己的云盘根目录
  - 自动处理重名

【在线预览】
  - 图片：JPG, PNG, GIF, WebP, BMP
  - 文档：PDF（浏览器内嵌）
  - 视频：MP4, WebM, OGG（HTML5 播放器）
  - 音频：MP3, WAV, OGG（HTML5 播放器）
  - 其他类型自动下载

【响应式设计】
  - 自适应 PC、平板、手机
  - 禁止双指缩放，保证界面稳定


三、环境要求
------------
Web 服务器：Apache / Nginx / 任何支持 PHP 的服务器
PHP 版本：>= 7.0（推荐 7.4+）
PHP 扩展：
  - json（默认开启）
  - session（默认开启）
  - fileinfo（可选，推荐用于识别文件类型）
  - zip（可选，用于批量下载）
目录权限：
  - uploads/ 目录必须可读可写（755 或 777）
  - 项目根目录需可写（用于创建 users.json 和 shares.json）
浏览器：现代浏览器（Chrome / Edge / Firefox / Safari）


四、安装步骤
------------
1. 将源码所有文件上传到您的 Web 目录（例如 public_html/cloud/）。
2. 设置权限：
   - 将 uploads/ 目录权限设为 755 或 777。
   - 确保项目根目录可写。
3. 访问：https://您的域名/项目目录/index.php
4. 默认管理员账号：admin    密码：admin
5. 首次登录后建议立即修改密码（通过注册新用户或手动修改 users.json）。


五、使用指南
------------
【注册/登录】
  - 点击“注册”创建新用户（密码至少 4 位）。
  - 登录后会话保持 30 天，关闭浏览器仍保持登录。

【上传文件】
  - 点击“上传文件”按钮，选择文件（支持多选）。
  - 进度条实时显示，自动处理重名。

【管理文件夹】
  - 点击文件夹名称进入子目录。
  - 右侧按钮可重命名或删除文件夹。

【分享文件】
  1. 点击文件行中的“分享”图标（fa-share-alt）。
  2. 选择有效期（1天/3天/7天/30天/永久）。
  3. 生成链接后复制即可分享。

【撤销分享】
  - 若文件已分享，分享图标变为红色叉号（fa-times-circle），点击即可撤销。

【复制链接（图床）】
  - 点击“复制链接”图标（fa-link）。
  - 若为图片，会询问是否复制为 Markdown 或纯链接。
  - 链接可直接用于 HTML、Markdown 等。

【批量操作】
  1. 勾选文件前的复选框（或点击“全选”）。
  2. 使用底部按钮执行批量操作：复制链接、删除、移动、下载（ZIP）。

【转存他人分享】
  - 打开他人分享链接（share.php?token=xxx）。
  - 若已登录且不是自己的文件，底部显示“转存到我的云盘”。
  - 点击后文件立即复制到您的根目录。

【查看存储空间】
  - 左侧栏“存储空间”显示您已使用的总大小。


六、文件结构说明
----------------
项目根目录/
├── config.php          # 系统配置（用户文件、目录、会话设置）
├── index.php           # 主程序（登录、文件管理、AJAX 接口）
├── login.php           # 登录页面
├── register.php        # 注册页面
├── share.php           # 分享链接访问入口
├── 404.php             # 自定义 404 错误页
├── .htaccess           # 安全设置（禁止目录列表、保护敏感文件）
├── uploads/            # 所有用户文件存放目录（自动按用户名建子目录）
│   ├── admin/          # 管理员文件
│   └── 其他用户/
├── users.json          # 用户数据（用户名 -> 密码哈希）
└── shares.json         # 分享记录（token -> 文件路径、用户、过期时间）


七、安全建议
------------
1. 修改默认密码：登录后注册新用户并删除 admin 账户（或修改其密码）。
2. 保护敏感文件：.htaccess 已禁止访问 users.json 和 shares.json，请勿删除。
3. 启用 HTTPS：系统会自动检测，但请确保服务器已配置 SSL 证书。
4. 定期备份：定期备份 uploads/、users.json 和 shares.json。
5. 限制上传大小：如需调整，修改 php.ini 或 .htaccess 中的 upload_max_filesize 等参数。


八、常见问题（FAQ）
-------------------
Q：上传大文件失败怎么办？
A：检查 php.ini 中的 upload_max_filesize 和 post_max_size，调整为所需大小。
   虚拟主机可联系客服或通过 .htaccess 设置（需主机支持）。

Q：复制链接后访问显示 404？
A：检查 config.php 中的 UPLOAD_BASE_URL 是否生成正确（通常自动生成）。
   若仍有问题，可手动修改该常量。

Q：如何重置管理员密码？
A：删除 users.json 文件，系统会自动重建并生成默认 admin/admin。
   注意：会丢失所有用户数据。建议通过注册新用户来重置。

Q：分享链接打不开？
A：检查 shares.json 是否存在且可写，查看 share.php 中的文件路径是否正确
   （确保包含用户名目录）。

Q：在线预览视频/音频失败？
A：确保浏览器支持该格式（推荐 MP4/H.264 视频、MP3 音频）。
   部分虚拟主机可能限制文件大小，请调整相关配置。

Q：可以自定义网站标题和 Logo 吗？
A：可以。修改 template.php 和 login.php 中的 <title> 以及导航栏图标。
